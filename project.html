Tennis Ball Speed AI Project - Spring 2024

  • I built a tennis ball speed AI model for my AI class project. The Model uses a connected camera to capture the ball in motion and to detect the displacement of the ball.
  • The speed is determined using displacement training data in a single variable linear regression calculation.

Temperature Sensor for Large Cooler - Summer 2024

  • I built an automated temperature sensor that uses a temperature sensor, a Raspberry Pi, and Python libraries to regulate the temperature of the large walk-in cooler at The Shoals Shack.
  • The sensor uses the Textbelt API to notify my manager when the cooler reaches a specified temperature by a text message.
